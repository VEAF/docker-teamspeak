# Simple Teamspeak docker stack

Note: free licence need the server to be updated every 6 months, just change the "TS_VERSION" in your .env file.

## install

```shell
git clone https://github.com/VEAF/docker-teamspeak.git teamspeak
cd teamspeak
```

## run / upgrade

```shell
make upgrade
```

## more commands

```shell
make
```
